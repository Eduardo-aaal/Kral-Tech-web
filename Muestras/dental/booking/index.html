Sistema de Agendamiento

Esta aplicación necesita que el servidor de Next.js esté ejecutándose. Sigue estos pasos para iniciarlo:

  1. Abre una terminal (CMD, PowerShell o Git Bash)
  2. Ve a la carpeta del proyecto:
    cd Muestras/dental/booking
  3. Ejecuta el servidor:
    npm run dev
  4. Espera a que aparezca "ready - started server on 0.0.0.0:3000"
  5. Haz clic en el botón de abajo 👇
📅 Ir al agendamiento

Una vez iniciado el servidor, el sistema estará disponible en
http://localhost:3000
